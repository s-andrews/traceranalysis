Tracer Analysis Tool

Data from tracer experiments using isotope labeled metabolites (eg 13C glucose) must be normalized to natural isotopologue distribution. This tool will quickly and reliably transform raw mass spec AUC values into the percent representation of each isotopologue measured.

1. Enter unlabeled data:

Enter the unlabeled data for your chosen metabolite in the box below. This data is used for normalisation.**

NOTE: Data should be entered as comma seperated values with each row on a new line. Paste in NUMERICAL VALUES ONLY. Do not paste in the column and row headers.

2. Enter your full data set:

Enter the data for your chosen metabolite in the box below. Use the same format as you did for the unlabeled data and make sure the number of columns is consistent.

Data formatting:

Data should be organised into separate columns for each isotopologue measured and separate rows for each replicate sample.

**It is recommended that you normalise to control data from samples that did not contain an isotope label. If you do not have unlabeled data, you can use theoretical isotopologue distribution values for normalisation.
